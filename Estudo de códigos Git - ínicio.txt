## :books: Aqui são alguns códigos que já aprendi :books:

![BMO](https://i.pinimg.com/originals/bc/97/e3/bc97e36907b3790f2a348d35299c03e4.gif)

*ls ou dir - mostra os arquivos dentro de pasta, ou pastas dentro de pastas*

*ctrl+L - limpa histórico*

*git status - olha os estágios do ciclo*

*mkdir - cria uma pasta*

*mv + nome da pasta - move arquivos para outras pastas*

*cd + pasta - se move para dentro da pasta mencionada*

*cd .. - retrai um nível*

*git add (nome do arquivo)- adiciona o arquivo no ciclo*

*git commit -m - add um comentário* 

*git add * - adiciona tudo ao stage*

*git remote add origin (link url) - bota para a area remota* 

*git remote -v - mostra o link cadastrado*

*git push origin master/main - puxa os arquivos para nuvem*

*colchetes [ ] md - coloque o texto e ( ) parenteses coloque o link*

*git clone + link https - clona do github para o git bash*

*git rm -r - deleta uma pasta ou arquivo*
